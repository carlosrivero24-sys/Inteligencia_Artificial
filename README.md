# Inteligencia_Artificial
_Este es mi primer repositorio donde fui guardando las actividades que nos daba el profesor Miguel Angel con el fin de guardarlos bien acomodados en carpetas y poder verlos y estudiarlos cuando lo necesite, ya que cada linea del codigo esta explicado con mis palabras para poder entender que fue lo que  hice. Podemos encontrar carpetas como CNN, RNN y Multicapa que son redes neuronales y varios ejemplos hechos en clase._

Carpeta de Perceptron: Aqui encontramos nuestras operaciones logicas NOT y OR, fue el primer modelo que usamos que fue para introducirnos a las redes neuronales 

Carpeta de Multicapa: Esta es un poco mas avanzada que la de perceptron, aqui aprendemos como agregar capas de aprendizaje, funciones de activacion, Etc.

Carpeta de CNN: En esta carpeta hay actividades con redes convolucionales, las que se usan para imágenes mas que nada. Incluye ejemplos sencillos, pruebas y algunos modelos entrenados.

Carpeta de RNN: Esta fue la ultima carpeta que vimos, aqui trabajamos con secuencias para predecir a futuro con los datos que ya tenemos, hay diferentes modelos con datos reales, hay uno que fue hecho por una compañera y yo que fue un experimento que salio algo regular pero se aprendio mas en base a eso. 

_Y eso es todo pueden revisarlo y si les sirve de algo los pueden ocupar._
